\section{日本語と数式}
\textbf{ゼータ関数}（zeta function）というのは
\begin{equation}                     % 数式中の漢字もOK
\zeta(s) \stackrel{定義}{=} \sum_{n=1}^\infty \frac{1}{n^s}
= \prod_{p\colon 素数}\frac{1}{1-p^{-s}}
\end{equation}
とかいう奴のこと．
